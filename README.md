Implement, Build and Release your Web App with Google Technologies
===================
 
 
## Necessary Prerequisites 
>- Linux (Ubuntu 16.04 in my  case)
>- Git
>- Curl
>- Docker
>- VirtualBox
>- Minikube
>- Google Cloud SDK (contains Kubectl)


## Not-Necessary Prerequisites 
>- Golang
>- NodeJS
>- Angular CLI
